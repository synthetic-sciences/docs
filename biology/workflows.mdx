Each recipe shows what to type into the agent and what comes out the other side. Use them as starting points and adapt to your data.

## Protein structure prediction

> Predict structures for the 12 sequences in `targets.fasta` with ESMFold. Rank by pLDDT, write the top three as PDBs, and align them against `template.pdb` to compare folds.

What happens:

1. The agent loads the sequences from FASTA.
2. Runs ESMFold across all 12 (parallelizing on GPU through Modal if available).
3. Computes pLDDT for each prediction.
4. Writes the top three to PDB and runs structural alignment against the template.
5. Reports RMSD and TM-score for each alignment.

## Variant calling pipeline

> Run a germline variant calling pipeline on these 50 paired-end FASTQ samples. Use BWA-MEM for alignment, GATK HaplotypeCaller for calling, and produce a joint-called VCF with annotations from VEP.

What happens:

1. Quality-checks the FASTQs with FastQC.
2. Aligns with BWA-MEM against the reference.
3. Marks duplicates and runs base quality score recalibration.
4. Calls variants per sample, then joint-genotypes across the cohort.
5. Annotates the resulting VCF with VEP for functional predictions.

## Single-cell clustering and trajectory

> Run a Scanpy analysis on this 10X dataset. Filter cells with fewer than 200 genes, normalize, find HVGs, run PCA, build neighbors, cluster with Leiden at resolution 1.0, and run a PAGA trajectory inference.

What happens:

1. Loads the 10X matrix into AnnData.
2. Filters cells and genes by quality metrics.
3. Normalizes (scran or shifted log) and identifies highly variable genes.
4. Runs PCA, neighborhood graph, and Leiden clustering.
5. Runs PAGA on the cluster graph and reports connectivities.
6. Generates UMAP and clustered heatmap for marker genes.

## Multi-omics integration

> Integrate this RNA-seq, ATAC-seq, and methylation dataset across 200 samples using MOFA+. Identify the top 5 latent factors and the genes/peaks/CpGs that drive each one.

What happens:

1. Loads each omics layer and aligns sample IDs.
2. Configures MOFA+ with the appropriate likelihood per layer.
3. Trains the model.
4. Reports variance explained per factor per layer.
5. Returns the top features driving each factor across all three views.

## Phylogenetic inference

> Build a maximum likelihood phylogenetic tree for these 200 SARS-CoV-2 spike protein sequences using IQ-TREE. Use the best-fit substitution model and 1000 bootstrap replicates.

What happens:

1. Aligns the sequences with MAFFT.
2. Runs ModelFinder to pick the best substitution model.
3. Runs IQ-TREE with the chosen model and 1000 ultrafast bootstrap replicates.
4. Returns the tree in Newick format with bootstrap supports.

## Differential expression with DESeq2

> Run differential expression on this RNA-seq count matrix comparing treated vs control. Use DESeq2 with apeglm shrinkage. Filter for padj less than 0.05 and absolute log2FC greater than 1.

What happens:

1. Loads the count matrix and metadata.
2. Builds the DESeq2 dataset with the design formula.
3. Runs DESeq2 normalization and Wald test.
4. Applies apeglm shrinkage to the log2 fold changes.
5. Filters and writes the significant gene list with shrunken effect sizes.

## Structure-based drug design

> Dock the FDA-approved drug library against this AlphaFold model of the target. Use AutoDock Vina with exhaustiveness 32, then re-score the top 100 hits with a deep learning scoring function.

What happens:

1. Loads the AlphaFold model and identifies likely binding pockets.
2. Prepares ligands from the FDA-approved library.
3. Runs Vina docking with the requested exhaustiveness.
4. Re-scores the top 100 with a learned scorer (GNINA or similar).
5. Returns the consensus-ranked list with poses.
