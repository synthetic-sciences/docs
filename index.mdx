A graph-based research workspace and a terminal agent for scientific work. Two products that ship independently and share a single account.

## Get going in one command

Install the CLI:

```bash
npx synci
```

Or open the research workspace at [atlas.syntheticsciences.ai](https://atlas.syntheticsciences.ai). Both share a Synthetic Sciences account.

## Atlas

A graph-based research workspace. Hypotheses, experiments, sources, and decisions live as durable, branchable nodes.

- **[Introduction](/introduction).** What Atlas is for and how it works.
- **[Quickstart](/quickstart).** Sign in, create your first node, index a source, talk to the agent.
- **[Web app tour](/web-app).** Every UI area explained.

## Synci CLI

A terminal agent for scientific work. The binary is `synci`. Pick a research mode and the agent loads the right domain conventions, tools, and backends for the work.

- **[Introduction](/cli/introduction).** What `synci` is and what each research mode covers.
- **[Installation](/cli/installation).** `npx synci` does the entire setup.
- **[Quickstart](/cli/quickstart).** Connect, pick a mode, send your first prompt.

### Research modes

| Mode | For |
|---|---|
| [Research](/cli/research/research) | Cross-disciplinary work, literature review, hypothesis generation, GPU compute orchestration |
| [Physics](/cli/research/physics) | Simulation, PDE solving, dynamical systems, symbolic regression |
| [Chemistry](/cli/research/chemistry) | Drug discovery, molecular design, ADMET, docking, structure prediction |
| [Biology](/cli/research/biology) | Bioinformatics, sequencing pipelines, single-cell analysis, structure prediction |
| [Write](/cli/research/write) | LaTeX papers, grant proposals, literature reviews with verified citations |
| [Flywheel](/cli/research/flywheel) | Build a specialized LLM from your production data |

## Common workflows

<AccordionGroup>
  <Accordion title="Research-only. Atlas on its own.">
    Start at the [Atlas quickstart](/quickstart). Create a root node, index a paper or two, talk to the agent, stage an experiment when you're ready. The web app at [atlas.syntheticsciences.ai](https://atlas.syntheticsciences.ai) is enough.
  </Accordion>

  <Accordion title="Coding-only. Synci CLI on its own.">
    Install with `npx synci`, then run `synci web --mode <your-domain>`. The bundled domain tools and credential sync mean you can start running scientific workflows in minutes. No Atlas account needed.
  </Accordion>

  <Accordion title="Both. Synci CLI runs the work, Atlas tracks the program.">
    Set up both products. The agent runs the experiments in your repo, while Atlas keeps the long-running research graph of what you tried and why.
  </Accordion>
</AccordionGroup>

## Account and billing

Both products share a Synthetic Sciences account. The Atlas web app uses your account for project access. The Synci CLI dashboard at [cli.syntheticsciences.ai](https://cli.syntheticsciences.ai) issues Bearer tokens, holds synced credentials, and tracks credit usage.

Plans and credits are scoped per product. Subscribe to either or both. Visit your account page for plan details.
