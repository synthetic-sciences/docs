Synthetic Sciences ships two products that work independently and pair naturally. **Thesis** is a graph-based research workspace where hypotheses, experiments, sources, and decisions live as durable, branchable nodes. **Synsci CLI** is a terminal AI coding agent built for ML work, with 83 bundled skills and native cloud sync. They use the same Synthetic Sciences account, and Synsci CLI can read and write Thesis state through MCP.

## Common workflows

<AccordionGroup>
  <Accordion title="Research-only — Thesis on its own">
    Start at [Thesis quickstart](/quickstart). Create a root node, index a paper or two, talk to the agent, stage an experiment when you're ready. The web app at [thesis.syntheticsciences.ai](https://thesis.syntheticsciences.ai) is enough — you don't need any CLI.
  </Accordion>

  <Accordion title="Coding-only — Synsci CLI on its own">
    Install with `npx synsci` (the all-in-one installer), then run `synsc web` to open the browser UI. The bundled skills and credential sync mean you can start running training scripts and managing cloud resources in minutes — no Thesis account needed.
  </Accordion>

  <Accordion title="Both — Synsci CLI authoring code, Thesis tracking the program">
    Set up both products. Connect them with `synsc mcp add thesis --url https://thesis.syntheticsciences.ai/mcp-server` so the terminal agent can read and write your research graph. The agent stages experiments in Thesis while it writes the actual training code in your repo. See [Connect Synsci CLI to Thesis](/cli/mcp).
  </Accordion>

  <Accordion title="Cursor or Claude Code + Thesis (no Synsci CLI)">
    Use your existing IDE agent host. Run `npx --yes @synsci/thesis setup` to wire Thesis into the host's MCP config. Your agent gets the 86 Thesis tools alongside its built-ins. Synsci CLI isn't required.
  </Accordion>
</AccordionGroup>

## Account and billing

Both products share a Synthetic Sciences account. Sign in once and:

- Thesis web app and MCP server use your account for project access.
- Synsci CLI dashboard at [cli.syntheticsciences.ai](https://cli.syntheticsciences.ai) issues Bearer tokens, holds synced credentials, and tracks credit usage.

Plans and credits are scoped per product. You can subscribe to one without the other, or both with a combined plan. Visit your account page for plan details.
